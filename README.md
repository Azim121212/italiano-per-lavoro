# IL - Landing Page

Лендинг для курса итальянского языка для студентов, которые хотят работать в Италии.

## Описание

Одностраничный сайт для продажи курса итальянского языка. Сайт собирает заявки через WhatsApp и Telegram, создает доверие и показывает, что курс решает проблему студентов в Мессине.

## Структура сайта

1. **Хедер** - Название "IL", слоган и кнопка CTA
2. **Проблема** - Описание проблемы студентов
3. **Решение** - Описание курса и подхода
4. **Пакет/Цены** - Базовый пакет 260€ за 3 месяца
5. **Программа курса** - 3 модуля обучения
6. **Отзывы** - Плейсхолдер для будущих отзывов
7. **Контакты** - Кнопки WhatsApp и Telegram
8. **Футер** - Копирайт

## Особенности

- ✅ Минималистичный дизайн
- ✅ Полностью адаптивный (мобильный, планшет, десктоп)

## Настройка контактов

**ВАЖНО:** Перед использованием замените контактные данные в `index.html`:

1. **WhatsApp**: Замените `1234567890` на ваш номер телефона с кодом страны (без +)
   - Формат: `https://wa.me/1234567890`
   - Пример: `https://wa.me/393912345678` (для итальянского номера)

2. **Telegram**: Замените `username` на ваш Telegram username
   - Формат: `https://t.me/username`
   - Пример: `https://t.me/italiano_per_lavoro`

Найдите в файле `index.html` все ссылки с `href="https://wa.me/..."` и `href="https://t.me/..."` и замените на ваши реальные контакты.

## Запуск локального сервера

### Быстрый запуск:

**macOS/Linux:**
```bash
./start-server.sh
```

**Windows:**
```cmd
start-server.bat
```

### Альтернативные способы:

**1. Через Python (рекомендуется):**
```bash
python3 -m http.server 8000
# или
python -m http.server 8000
```

**2. Через Node.js:**
```bash
npx http-server
```

**3. Прямое открытие:**
Просто откройте `index.html` в браузере (некоторые функции могут не работать из-за CORS)

### После запуска:

Откройте в браузере: **http://localhost:8000**

Сервер будет работать до тех пор, пока вы не остановите его (Ctrl+C в терминале).
