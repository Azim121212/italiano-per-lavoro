Виртуальная платформа обучения для курса "Italiano per lavoro".

## Доступ

**URL:** `/platform/index.html`

## Функционал

### Для студента

1. **Мои курсы**
   - Просмотр доступных курсов
   - Прогресс по курсу
   - Доступ к урокам и заданиям

2. **Прогресс**
   - Общий прогресс по курсу (в процентах)
   - Количество завершённых уроков
   - Общее количество очков
   - Средний балл

3. **Оценки**
   - История всех оценок
   - Комментарии преподавателя
   - Очки за каждое задание

4. **Рейтинг**
   - Рейтинг группы по очкам
   - Позиция студента в группе

### Для преподавателя

1. **Мои группы**
   - Просмотр всех групп преподавателя
   - Список студентов в группе
   - Просмотр прогресса студентов

2. **Уроки**
   - Просмотр всех уроков курса
   - Информация о заданиях

3. **Задания**
   - Просмотр всех заданий студентов
   - Оценка заданий
   - Добавление комментариев
   - Фильтрация по статусу (требует оценки / оценено)

4. **Статистика**
   - Общая статистика по группам
   - Средний балл группы
   - Средний прогресс группы

## Структура уроков

Каждый урок включает:

1. **Теория** - текстовый материал с ключевыми моментами
2. **Задания** - разные типы:
   - Текстовые задания (вставка ответа)
   - Множественный выбор (quiz)
   - Интерактивные игры (викторины типа Kahoot)
   - Практические кейсы

3. **Прогресс** - шкала % выполнения урока
4. **Очки** - за каждое задание можно получить очки
5. **Обратная связь** - преподаватель комментирует ответы

## Первый месяц обучения

Платформа включает 12 уроков первого месяца:

1. Основы повседневного итальянского
2. Работа с клиентами
3. Числа, время, даты
4. Магазины и заказы
5. Работа с документами
6. Общение с коллегами
7. Повседневные ситуации
8. Собеседование
9. Работа с заказами
10. Деловая переписка
11. Грамматика для работы
12. Итоговый урок

Каждый урок содержит:
- Теоретический материал
- 2-3 задания разных типов
- 10 очков (итоговый урок - 20 очков)
- Всего за месяц: 130 очков

## Система оценок

- **Автоматическая проверка**: Quiz и интерактивные задания проверяются автоматически
- **Ручная проверка**: Текстовые задания проверяются преподавателем
- **Обратная связь**: Преподаватель может добавить комментарий к каждому заданию
- **Очки**: За каждое задание можно получить определённое количество очков

## Разработка

Для разработки и тестирования:

1. Откройте `platform/index.html` в браузере
2. Войдите с учетными данными для тестирования
3. Начните работу с платформой
